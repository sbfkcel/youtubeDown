# youtubeDown

Electron based Youtube video download tool

## Use

2.1 download `https://github.com/sbfkcel/youtubeDown.git`

2.2 `npm install`

2.3 `npm run start`

## Compile

windows：
```bash
npm run build-win64
```

Mac OS：
```bash
npm run build-mac
```
